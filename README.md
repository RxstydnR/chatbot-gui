## Streamlit GUI誘導型AIチャットボット — インタラクティブに会話できるAIアシスタント
Streamlit上で動作する、GUI誘導型のAIチャットボット。ユーザーは、エージェントが描画するGUIを操作だけで直感的かつ簡単に対話できるデモアプリ。
LangGraphで実装されたエージェントがTool Callを通じてUI（選択肢、フォーム入力、ボタンなど）を動的に描画することで会話を進めていく。

### Streamlit GUI-Guided AI Chatbot — An Interactive AI Assistant
A GUI-guided AI chatbot running on Streamlit. This demo app lets users interact intuitively and easily by simply operating the GUI rendered by the agent. An agent built with LangGraph advances the conversation by dynamically rendering UI elements (choices, form inputs, buttons, etc.) via tool calls.
